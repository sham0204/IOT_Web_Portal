# SmartDrishti IoT Learning Platform - Project Summary

## Overview
SmartDrishti is a comprehensive IoT learning platform that enables users to learn IoT development through guided, hands-on projects with real-time monitoring capabilities.

## Key Features
- **Project Management**: Create, organize, and track IoT learning projects
- **Step-by-Step Guidance**: Detailed instructions with rich media support
- **Real-Time Monitoring**: Live sensor data visualization from connected devices
- **IoT Integration**: Support for ESP32, Arduino, Raspberry Pi and various sensors
- **User Management**: Secure authentication and profile management

## Technology Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express.js, Socket.IO, MQTT
- **Database**: SQLite (local), PostgreSQL (production)
- **Real-time**: WebSocket communication for live updates

## Core Components
- Interactive project creation and management
- Step-by-step guided tutorials with media support
- Live monitoring dashboard with real-time charts
- Device registration and sensor data handling
- User authentication and profile management

## Value Proposition
Bridges the gap between IoT theory and practical implementation by providing a structured learning environment with immediate feedback from real hardware devices, accelerating IoT skill development.

## Target Users
- Educational institutions for IoT curriculum
- Individual learners seeking hands-on experience
- Corporate training programs
- IoT hobbyists and enthusiasts
